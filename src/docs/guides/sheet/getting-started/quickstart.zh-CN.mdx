## Univer 开源版

### 从模板开始

如果你想在本地搭建 Univer 应用，可以使用以下命令从模板创建一个新的 Univer 应用。

<Tabs items={['pnpm', 'npm']}>
  <Tabs.Tab label="pnpm">
    执行以下命令需要设备安装有 `nodejs >= 18`, `pnpm >= 7` 和 `git`。

    ```shell
    pnpm dlx degit dream-num/univer-sheet-start-kit univer-sheet-start-kit
    cd univer-sheet-start-kit
    pnpm install
    pnpm dev
    ```
  </Tabs.Tab>
  <Tabs.Tab label="npm">
    执行以下命令需要设备安装有 `nodejs >= 18`, `npm >= 8` 和 `git`。

    ```shell
    npx degit dream-num/univer-sheet-start-kit univer-sheet-start-kit
    cd univer-sheet-start-kit
    npm install
    npm run dev
    ```
  </Tabs.Tab>
</Tabs>

至此，你已经成功创建了属于你的第一个 Univer 项目。你可以在 [dream-num/univer-sheet-start-kit](https://github.com/dream-num/univer-sheet-start-kit) 找到该模板的源代码。

## 集成 Univer 高级功能

当然，你有可能不满足于此，想要打印、导入导出、协同编辑等更多功能，这些功能需要更多的配置以及服务端的支持。不过不用担心，Univer 也为你提供了快速体验这些功能的方式，不妨照着下面的步骤来试试：

<Steps>

