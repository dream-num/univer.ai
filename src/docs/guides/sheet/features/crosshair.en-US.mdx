# Crosshair Highlight

<BadgeGroup values={[UniverTypes.SHEET]} value={UniverTypes.SHEET} />

<Sandbox entryFile={entryFile} />

Crosshair highlight highlights the row and column of the selected cell to make it easier for users to identify the currently selected cell.

## Installation

<Tabs items={['pnpm', 'npm']}>
  <Tabs.Tab label="pnpm">
    ```shell
    pnpm add @univerjs/sheets-crosshair-highlight
    ```
  </Tabs.Tab>
  <Tabs.Tab label="npm">
    ```shell
    npm install @univerjs/sheets-crosshair-highlight
    ```
  </Tabs.Tab>
</Tabs>

## Import

```typescript
import '@univerjs/sheets-crosshair-highlight/lib/index.css';

import { UniverSheetsCrosshairHighlightPlugin } from '@univerjs/sheets-crosshair-highlight';
```

## Internationalization

```typescript
import { LocaleType, Tools } from '@univerjs/core';
import SheetsCrosshairHighlightEnUS from '@univerjs/sheets-crosshair-highlight/locale/en-US';

const univer = new Univer({
  theme: defaultTheme,
  locale: LocaleType.EN_US,
  locales: {
    [LocaleType.EN_US]: Tools.deepMerge(
      SheetsCrosshairHighlightEnUS 
    ),
  },
});
```

## Register

```typescript
univer.registerPlugin(UniverSheetsCrosshairHighlightPlugin);
```
