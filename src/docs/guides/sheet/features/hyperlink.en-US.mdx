# HyperLink

<BadgeGroup values={[UniverTypes.SHEET, UniverTypes.DOC]} value={UniverTypes.SHEET} />

Insert hyperlinks, support click to jump to specified link, cell, or worksheet.

## Installation

<Tabs items={['pnpm', 'npm']}>
  <Tabs.Tab label="pnpm">
    ```shell
    pnpm add @univerjs/sheets-hyper-link @univerjs/sheets-hyper-link-ui
    ```
  </Tabs.Tab>
  <Tabs.Tab label="npm">
    ```shell
    npm install @univerjs/sheets-hyper-link @univerjs/sheets-hyper-link-ui
    ```
  </Tabs.Tab>
</Tabs>

## Import

```typescript
import '@univerjs/sheets-hyper-link-ui/lib/index.css';

import { UniverSheetsHyperLinkPlugin } from '@univerjs/sheets-hyper-link';
import { UniverSheetsHyperLinkUIPlugin } from '@univerjs/sheets-hyper-link-ui';
```

## Internationalization

```typescript
import { LocaleType, Tools } from '@univerjs/core';
import SheetsHyperLinkUIEnUS from '@univerjs/sheets-hyper-link-ui/locale/en-US';

const univer = new Univer({
  theme: defaultTheme,
  locale: LocaleType.EN_US,
  locales: {
    [LocaleType.EN_US]: Tools.deepMerge(
      SheetsHyperLinkUIEnUS
    ),
  },
});
```

## Register

```typescript
univer.registerPlugin(UniverSheetsHyperLinkPlugin);
univer.registerPlugin(UniverSheetsHyperLinkUIPlugin);
```
