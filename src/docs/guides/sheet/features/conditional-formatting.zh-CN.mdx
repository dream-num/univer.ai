# 条件格式

<BadgeGroup values={[UniverTypes.SHEET]} value={UniverTypes.SHEET} />

条件格式是一种在单元格中设置规则，以根据数据的值自动设置单元格格式的功能。

## 安装

<Tabs items={['pnpm', 'npm']}>
  <Tabs.Tab label="pnpm">
    ```shell
    pnpm add @univerjs/sheets-conditional-formatting @univerjs/sheets-conditional-formatting-ui
    ```
  </Tabs.Tab>
  <Tabs.Tab label="npm">
    ```shell
    npm install @univerjs/sheets-conditional-formatting @univerjs/sheets-conditional-formatting-ui
    ```
  </Tabs.Tab>
</Tabs>

## 引入

```typescript
import '@univerjs/sheets-conditional-formatting-ui/lib/index.css';

import { UniverSheetsConditionalFormattingPlugin } from '@univerjs/sheets-conditional-formatting';
import { UniverSheetsConditionalFormattingUIPlugin } from '@univerjs/sheets-conditional-formatting-ui';
```

## 国际化

```typescript
import { LocaleType, Tools } from '@univerjs/core';
import SheetsConditionalFormattingUIZhCN from '@univerjs/sheets-conditional-formatting-ui/locale/zh-CN';

const univer = new Univer({
  theme: defaultTheme,
  locale: LocaleType.ZH_CN,
  locales: {
    [LocaleType.ZH_CN]: Tools.deepMerge(
      SheetsConditionalFormattingUIZhCN
    ),
  },
});
```

## 注册

```typescript
univer.registerPlugin(UniverSheetsConditionalFormattingPlugin);
univer.registerPlugin(UniverSheetsConditionalFormattingUIPlugin);
```
