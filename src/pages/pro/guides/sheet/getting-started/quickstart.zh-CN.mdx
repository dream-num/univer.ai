## 从模板创建

Univer Pro 的大部分功能都依赖服务端，如果你想进行自有部署体验或定制化开发，我们推荐用 Docker 快速部署服务端，具体步骤如下：

```bash
git clone -b v0.1.11 https://github.com/dream-num/helm-charts --depth=1
cd helm-charts/docker-compose && ./run.sh
```

服务端部署后，可以使用以下命令从模板创建一个新的 Univer Pro 前端应用。

<Tabs items={['pnpm', 'npm']}>
  <Tabs.Tab label="pnpm">
    ```shell
    pnpm dlx degit dream-num/univer-pro-sheet-start-kit [your-app-name]
    cd [your-app-name] && pnpm install # pnpm version >= 8
    pnpm dev
    ```
  </Tabs.Tab>
  <Tabs.Tab label="npm">
    ```shell
    npx degit dream-num/univer-pro-sheet-start-kit [your-app-name]
    cd [your-app-name] && npm install # npm version >= 7
    npm run dev
    ```
  </Tabs.Tab>
</Tabs>

至此，你已经成功创建了属于你的第一个 Univer Pro 项目，您可以访问 [http://localhost:5173/](http://localhost:5173/) 体验 Univer Pro 的一些基本功能。
