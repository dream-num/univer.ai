We provide a command line tool [`@univerjs/create-cli`](https://github.com/dream-num/cli) for users who want to extend the functionality of Univer. You can refer to the following instructions to use it.

## Create a project

import { Tabs, TabItem } from "@astrojs/starlight/components";

    ```shell
      pnpm create @univerjs/cli@latest
    ```
  </TabItem>
  <TabItem label="yarn">
    ```shell
      yarn create @univerjs/cli@latest
    ```
  </TabItem>
