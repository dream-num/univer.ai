在 Univer 当中，除了使用底层的 API 之外，你还可以使用 `@univerjs/facade` 提供的 Facade API，它是对底层 API 的封装，更加简单易用。

你可以通过阅读[这篇文档](/guides/facade)了解如何使用 Facade API。

## 依赖的插件

本插件运行依赖以下插件：

- @univerjs/core
- @univerjs/engine-formula
- @univerjs/network
- @univerjs/sheets
- @univerjs/sheets-formula
- @univerjs/sheets-numfmt
