---
title: 自定义插件
---

我们为想要为 Univer 扩展功能的用户提供了命令行工具 [`@univerjs/create-cli`](https://github.com/dream-num/cli)，你可以参考下面的说明来使用。


## 创建项目

import { Tabs, TabItem } from "@astrojs/starlight/components";

<Tabs>
  <TabItem label="npm">
    ```shell
      npm create @univerjs/cli@latest
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```shell
      pnpm create @univerjs/cli@latest
    ```
  </TabItem>
  <TabItem label="yarn">
    ```shell
      yarn create @univerjs/cli@latest
    ```
  </TabItem>
</Tabs>
