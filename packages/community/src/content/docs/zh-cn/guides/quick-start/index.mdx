## 从模板创建

如果你想在本地搭建 Univer 应用，可以使用以下命令从模板创建一个新的 Univer 应用。我们提供了两个模板，分别用于创建 Univer Sheet 应用和 Univer Doc 应用。

import { Tabs, TabItem } from "@astrojs/starlight/components";

<Tabs>
  <TabItem label="npm">
    ```shell
      # 📊 create a univer sheet app
      npx degit dream-num/univer-sheet-start-kit [your-app-name]

      # 📝 create a univer doc app
      npx degit dream-num/univer-doc-start-kit [your-app-name]
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```shell
      # 📊 create a univer sheet app
      pnpm dlx degit dream-num/univer-sheet-start-kit [your-app-name]

      # 📝 create a univer doc app
      pnpm dlx degit dream-num/univer-doc-start-kit [your-app-name]
    ```
  </TabItem>
</Tabs>

至此，你已经成功创建了属于你的第一个 Univer 项目，并且体验了 Univer 的一些功能。
