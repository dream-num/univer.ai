# Univer Pro Sheet API

## Import XLSX

### Import XLSX To Get Unit Id

In a collaborative environment, each sheet has a `UnitId`. Use the API `importXLSXToUnitId` with the `file` parameter to obtain the `UnitId`, which can be used to access the sheet. The `file` parameter can be a `File` object or the `URL` of a remote file.

```javascript
univerAPI.importXLSXToUnitId(file).then((id)=>{
    console.log(id)
})
```

### Import XLSX To Get Workbook Data

If not in a collaborative environment, when using the import XLSX functionality individually, use the API `importXLSXToSnapshot` to return the sheet data in the [IWorkbookData](/api/core/interfaces/IWorkbookData.html) format.

```javascript
univerAPI.importXLSXToSnapshot(file).then((data)=>{
    console.log(data)
})
```

## Export XLSX

### Export XLSX by Unit Id

Use the API `exportXLSXByUnitId` with the `UnitId` parameter to return a `File` object.

```javascript
univerAPI.exportXLSXByUnitId(UnitId).then((file)=>{
    console.log(file)
})
```

### Export XLSX by Workbook Data

Use the API `exportXLSXBySnapshot` with the sheet data in `IWorkbookData` format to return a `File` object.

> How to [Get Workbook Data](/guides/sheet/facade/sheet#getting-workbook-data)

```javascript
univerAPI.exportXLSXBySnapshot(snapshot).then((file)=>{
    console.log(file)
})
```
