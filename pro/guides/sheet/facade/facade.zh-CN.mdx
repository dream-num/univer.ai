# 使用 Pro Facade API

您可以通过 Univer Pro Facade API 简单快速的使用 Univer Pro 提供的功能，Pro Facade API 完全基于 Univer Facade API 拓展而来，这意味着如果您从 Univer 升级到 Univer Pro 几乎不需要改变已有的代码。

### 预备知识

在开始之前，我们希望你已经熟悉了 Univer 的 [Facade API](/guides/sheet/facade/facade) 。如果你还没有学习过，我们强烈建议你先去了解一下，这将帮助你更好地理解本教程的内容。

相比于 Univer 的 Facade API，Univer Pro 的 Facade API 提供了 Pro 版本的功能，例如协同编辑、导入导出等。

## 安装

为了使用 Univer Pro 的 Facade API，你需要先安装它。以下是安装步骤：

使用你的包管理器安装：

```bash
# 使用 npm
npm install @univerjs-pro/facade

# 使用 pnpm
pnpm add @univerjs-pro/facade
```

## 使用

@univerjs/facade 提供的 API 是对 `Univer` 实例的封装，因此你需要在创建了 `Univer` 实例之后，再用 `FUniver` 包裹这个实例：

```typescript
import { FUniver } from "@univerjs-pro/facade";

const univerAPI = FUniver.newAPI(univer);
```

然后你就可以通过调用 `univerAPI` 的方法来使用 Univer 和 Univer Pro 的功能。

对于 Univer 用户如何升级到 Univer Pro，只需要改变导入的包名即可：

```diff
- import { FUniver } from "@univerjs/facade";
+ import { FUniver } from "@univerjs-pro/facade";
```
