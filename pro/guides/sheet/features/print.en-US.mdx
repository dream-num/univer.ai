# Print

`@univerjs-pro/sheets-print` provides high-precision printing functions, supporting print preview and print settings.

## Installation

<Tabs items={['pnpm', 'npm']}>
  <Tabs.Tab label="pnpm">
    ```shell
    pnpm add @univerjs-pro/sheets-print
    ```
  </Tabs.Tab>
  <Tabs.Tab label="npm">
    ```shell
    npm install @univerjs-pro/sheets-print
    ```
  </Tabs.Tab>
</Tabs>

## Import

```typescript
import '@univerjs-pro/sheets-print/lib/index.css';

import { SheetsPrintPlugin } from '@univerjs-pro/sheets-print'
```

## Internationalization

```typescript
import { LocaleType, Tools } from '@univerjs/core'
import { SheetsPrintPluginEnUS } from '@univerjs-pro/sheets-print/locale/en-US';

const univer = new Univer({
  theme: defaultTheme,
  locale: LocaleType.EN_US,
  locales: {
    [LocaleType.EN_US]: Tools.deepMerge(
      SheetsPrintPluginEnUS
    ),
  },
});
```

## Register

```typescript
univer.registerPlugin(SheetsPrintPluginEnUS);
```
